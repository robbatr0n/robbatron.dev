Built for my football team and used to track our stats and games throughout the season.

## Features

- Real-time match tagging — record goals, assists, tackles, interceptions and more as the game unfolds
- Player profiles with a full statistical breakdown across all appearances
- Complete match history with a per-game event log
- Attacking and defending leaderboards ranked across the season
- Dedicated goalkeeper tracking — saves, clean sheets, and a separate leaderboard
- End of season awards automatically computed from accumulated data

## Stack

Responsive frontend built with React, TypeScript and Tailwind. Supabase backend with a postgres database. Deployed on Vercel.
