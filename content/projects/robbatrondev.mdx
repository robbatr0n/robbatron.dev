## Features

- Minimalist design
- Project and lab sections with status indicators
- Individual project pages with image galleries
- MDX-based content — each project is a markdown file with frontmatter metadata

## Stack

Built with Next.js app router and TypeScript. Tailwind for styling, MDX with
next-mdx-remote for project content. Deployed on Vercel.
